① 正比的概念 — y = kx

定義

設 x 與 y 為兩個數量，k 是不為 0 的常數。
當 x 值改變時，y 值也隨之改變，且 y 值恆為 x 值的 k 倍，
即 y = kx（k ≠ 0）
滿足此關係式，稱 y 與 x 成正比。

📝 形成性評量

【記憶】下列哪個關係式表示 y 與 x 成正比？

【理解】若 y 與 x 成正比，當 x 增加為原來的 5 倍，則 y 變成原來的幾倍？

👨‍🏫 教師提示

  • 強調 k ≠ 0，k 為正則兩者同向變化，k 為負則反向
  • y = x² 不是正比（y 不是 x 的固定倍數）
  • xy = 6 是反比，別讓學生混淆

② 判別兩量是否成正比

判斷步驟

  1. 寫出 y 與 x 的關係式
  2. 檢查關係式能否寫成 y = kx（k ≠ 0） 的形式
  3. 若能，y 與 x 成正比；若不能，則不成正比

🎮 互動：正比 vs 非正比比較

📝 形成性評量

【記憶】以 x 為邊長的正方形，周長為 y，y 與 x 的關係式為何？y 與 x 是否成正比？

【理解】小明和姐姐歲數相差 6 歲，小明 x 歲，姐姐 y 歲，y 與 x 是否成正比？

👨‍🏫 教師提示

  • 常見錯誤：「兩量一起增加就是正比」——讓學生代入驗證比值是否固定
  • 列表法：若 y/x 比值固定，則成正比
  • 正三角形周長 y=3x（正比）vs 正方形面積 y=x²（非正比）—— 好的對照例

③ 列出正比關係式

解題步驟

  1. 設 y = kx（k ≠ 0）
  2. 將已知的 (x, y) 代入，求出 k
  3. 寫出完整關係式，再代入求解

📝 形成性評量

【理解】已知 y 與 x 成正比，x = 3 時 y = 15，則比例常數 k = ？

【應用】承上（k=5），當 x = −4 時，y = ？

👨‍🏫 教師提示

  • k 可以是負數！不要讓學生認為 k 一定是正的
  • 驗證：將 k 代回關係式，再用另一組數值驗算

④ 反比的概念 — xy = k

定義

設 x 與 y 為兩個數量，k 是不為 0 的常數。
當 x 值改變時，y 值也隨之改變，且 x 與 y 的乘積恆為定值，
即 xy = k（k ≠ 0）
滿足此關係式，稱 y 與 x 成反比。

拖曳滑桿改變 k 值，觀察雙曲線位置

📝 形成性評量

【記憶】xy = k（k≠0），當 k = 12，x = 3，則 y = ？

【理解】若 y 與 x 成反比，當 x 變成原來的 3 倍，y 變成原來的？

👨‍🏫 教師提示

  • 與正比的關鍵差異：正比看「比值 y/x」是否固定；反比看「乘積 xy」是否固定
  • 迷思：「x 增加時 y 減少」≠ 反比（k 為負時 x 增加 y 也增加！）

⑤ 判別兩量是否成反比

判斷步驟

  1. 寫出 y 與 x 的關係式
  2. 檢查關係式能否寫成 xy = k（k ≠ 0）
  3. 若能，y 與 x 成反比；若不能，則不成反比

🎮 互動：乘積檢驗器

📝 形成性評量

【記憶】36 個糖果均分給 x 人，每人 y 顆，y 與 x 成？

【理解】面積為 60 的三角形，底為 x，高為 y，則 y 與 x 成？

👨‍🏫 教師提示

  • 「均分」情境幾乎都是反比；「加總固定」情境幾乎都不是反比
  • 三角形面積題需先推導 xy = 2×面積，才確認是反比
  • y = 100 − x 這種「找零」情境：x+y=100 不是反比！

解題步驟

  1. 設 xy = k（k ≠ 0）
  2. 將已知的 (x, y) 代入，求出 k
  3. 寫出完整關係式，再代入求解

📝 形成性評量

【理解】已知 y 與 x 成反比，x = 6，y = −2，則 xy = ？k = ？

【應用】背完 1200 個英文單字，每天背 x 個，需要 y 天，若每天背 30 個，需幾天？

👨‍🏫 教師提示 + 迷思診療

  • 迷思：「反比」≠「x 增加 y 就減少」（k 為負時相反）
  • 用 xy=12 vs xy=−12 比較：xy=12 時 x 增加 y 減少；xy=−12 時 x 增加 y 也增加
  • 生活情境：oil price x × oil amount y = 1000（fixed budget）→ 反比

📋 重點整理 Cheat Sheet

✦ 正比

y = kx（k ≠ 0）
  • y 值 = x 值 × k（固定倍數）
  • 圖形：過原點的直線
  • 判別：y ÷ x 比值是否固定
  • 求 k：代入已知 (x, y) 求解
  • x 增 k 倍 → y 也增 k 倍

✦ 反比

xy = k（k ≠ 0）
  • x × y = 固定乘積 k
  • 圖形：雙曲線（不過原點）
  • 判別：x × y 乘積是否固定
  • 求 k：代入已知 (x, y) 求解
  • x 增 k 倍 → y 縮 1/k 倍（k>0 時）

✗ 常見錯誤

  • y = x + 5 ≠ 正比（有常數項）
  • y = x² ≠ 正比（指數不是 1）
  • x + y = 常數 ≠ 反比
  • 「y 隨 x 減少」≠ 反比（k 為負時不成立）

✓ 判別口訣

  • 正比：比值固定（y/x = k）
  • 反比：乘積固定（xy = k）
  • 「均分」→ 反比
  • 「含量百分比」→ 正比
  • 「找零」x+y=定值 → 都不是！
