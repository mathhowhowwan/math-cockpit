2-2 二元一次方程式的圖形

📐 核心概念

二元一次方程式
含兩個未知數，各為一次
如 ax + by = c
圖形為直線
所有解的點連成一條
無限延伸的直線
x 截距
令 y = 0 求 x
直線與 x 軸的交點
y 截距
令 x = 0 求 y
直線與 y 軸的交點

⚠️ 特殊直線

平行 x 軸：y = k（水平線，b=0 時 x 軸）
平行 y 軸：x = h（垂直線，a=0 時 y 軸）
過原點：令 x=0 → y=0，即 c = 0 時圖形過原點

3-1 比例式

📐 核心概念

比
a : b（b ≠ 0）
表示兩數的倍數關係
= a/b
比例式
a : b = c : d
外項：a、d
內項：b、c
比例式性質
外項乘積 = 內項乘積
ad = bc
連比
a : b : c
需統一中間項
才能合併

⚠️ 注意

比例式 a:b = c:d 等同 a/b = c/d，交叉相乘得 ad = bc
連比 a:b = 2:3，b:c = 3:5，中間項 b 已統一 → a:b:c = 2:3:5
若中間項不同，需先通分（取 LCM）再合併

3-2 正比與反比

📐 核心概念

正比
y = kx（k ≠ 0）
y/x = 常數
圖形：過原點直線
反比
y = k/x（k ≠ 0）
xy = 常數
圖形：雙曲線
判斷方式
正比：y ÷ x 固定
反比：y × x 固定
比例常數 k
正比：k = y ÷ x
反比：k = x × y

⚠️ 注意

正比圖形一定過原點；反比圖形（雙曲線）不過原點
距離固定時，時間與速度成反比；速度固定時，距離與時間成正比

4-1 一元一次不等式及其解

📐 四種不等符號

> 大於
「超過」「多於」
不含端點
< 小於
「不足」「未滿」「不到」
不含端點
≥ 大於等於
「至少」「不小於」「以上」
含端點
≤ 小於等於
「至多」「不超過」「以下」
含端點

📐 數線圖示

含端點（≥ 或 ≤）→ 實心圓；不含端點（> 或 <）→ 空心圓
大於類（> 或 ≥）→ 向右畫；小於類（< 或 ≤）→ 向左畫

⚠️ 易錯辨析

「超過」≠「以上」：超過不含本身（>），以上含本身（≥）
「不小於」= 大於或等於 = ≥，容易誤寫成 <
