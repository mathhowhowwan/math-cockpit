1不等符號的意義

【Q1】「小楷的身高 x 公分，身高不足 130 公分」，用不等式表示為：
【Q2】「至少 18 歲」用不等式表示（年齡為 x 歲）：

📌 教學提示

  • 強調「超過」≠「以上」：超過不含本身（>），以上含本身（≥）
  • 「未滿」「不到」「不足」對應 <；「以下」「至多」對應 ≤
  • 常見錯誤：學生常把「不小於」誤認為 <
  • 可用身高限制情境（遊樂設施）讓學生直觀理解

📌 教學提示

  • 步驟：設未知數 → 找數量關係 → 選不等符號 → 列式
  • 重點：「超過」才用 >，「不超過」用 ≤

3一元一次不等式的定義與解

【Q5】下列哪一個是不等式 3x + 2 ≤ 14 的解？
【Q6】不等式 2x − 1 < 5，下列哪些不是它的解？

📌 教學提示

  • 強調：不等式的解是一個範圍，不是一個點（連結下面數線單元）
  • 驗算口訣：代入 → 計算 → 判斷（成立或不成立）
  • 常見錯誤：學生只驗算整數，忽略分數或小數也可能是解
  • 對照方程式：方程式解唯一，不等式解為一個集合

📌 教學提示

  • 複合不等式 a < x < b 代表「x 同時大於 a 且小於 b」（AND）
  • 強調：左右兩端各自判斷含不含端點
  • 常見錯誤：同學以為兩端要相同（都實心或都空心）

📋 總複習 Cheat Sheet

不等符號對照

>大於、超過、高於
<小於、未滿、不足、不到
≥不小於、至少、以上（含）
≤不大於、至多、以下（含）

一元一次不等式

定義一種未知數、最高次方為 1
解代入使不等式成立的數
驗算代入 → 計算 → 判斷成立否

數線圖示規則

含端點（≥, ≤）實心圓 ●
不含端點（>, <）空心圓 ○
> 或 ≥向右延伸
< 或 ≤向左延伸
複合不等式兩端分別判斷

常見錯誤

超過 vs 以上> vs ≥
空心 vs 實心含本身才實心
複合兩端各自獨立判斷
解的範圍不只有整數解

  • 符號混淆：「超過」與「以上」混用（> vs ≥），需反覆強化「含本身」的概念
